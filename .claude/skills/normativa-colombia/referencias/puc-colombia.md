# Plan Único de Cuentas (PUC) Colombia

El PUC para comerciantes está regulado por el Decreto 2650 de 1993. Aunque con NIIF se permite flexibilidad, las empresas colombianas siguen usando estructuras heredadas del PUC para reportes oficiales (DIAN, Supersociedades, Cámara de Comercio).

## Estructura general (1 dígito)

| Clase | Naturaleza | Grupo |
